## Слайд 9 — Архитектура

**Заголовок:** Техническая архитектура

**Схема компонентов:**
```
[React + Vite]  →  [Node.js / Express API]  →  [PostgreSQL]
			      ↓
		      [YandexGPT API]
		      Yandex Cloud
```

**Почему этот стек:**
- **React + Vite** — быстрая разработка, адаптивность, единый язык с бэкендом
- **Node.js + PostgreSQL (Yandex Cloud)** — доступно из РФ, всё в одном облаке с YandexGPT
- **YandexGPT API** — глубокое понимание русского языка, быстрый API, скоринг за 3 сек

> ⚠️ API-ключ YandexGPT хранится только на сервере, клиент его не видит
