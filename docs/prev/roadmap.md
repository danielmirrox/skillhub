## Фаза 1 — MVP на хакатоне
**21–22 марта 2026 · 48 часов**

**Цель: рабочий прототип, который можно показать и потрогать**

| Часы | Что строим | Результат |
|------|-----------|-----------|
| 0–6 | БД + авторизация GitHub / Telegram | Пользователь может войти |
| 6–12 | Интеграция YandexGPT + промпт-инжиниринг | Работающий AI-скоринг |
| 12–18 | Профиль с рейтингом + лента пользователей | Можно смотреть чужие профили |
| 18–24 | Фильтр по рейтингу + эмуляция Captain PRO | Рабочий мэтчинг |
| 24–36 | Полировка UI, тестовые данные | Красивое демо |
| 36–48 | Подготовка питча, финальные правки | Готово к защите |

**MVP включает:**
- Регистрация через GitHub / Telegram
- Заполнение профиля (стек, опыт, ссылки)
- AI-скоринг через YandexGPT (0–100)
- Лента участников с фильтрами
- Эмуляция Captain PRO (без реальной оплаты)

## Фаза 2 — Валидация
**Март – Апрель 2026 · ~4 недели**

**Цель: проверить гипотезы, найти первых реальных пользователей**

- Запуск в 2–3 тематических Telegram-каналах хакатонов
- Сбор обратной связи от первых 50 пользователей
- Проверка гипотезы: **65% команд страдают от несоответствия скиллов**
- Замер конверсии Free → PRO
- Фикс критичных багов по фидбеку

**Целевые метрики фазы:**
- 50+ зарегистрированных профилей
- NPS > 40 среди первых пользователей
- Хотя бы 1 команда, сформированная через платформу

---

## Фаза 3 — Рост
**Апрель – Июнь 2026 · ~2 месяца**

**Цель: добавить ценность, начать монетизацию**

- **GitHub API** — автоматический анализ репозиториев для скоринга (меньше ручного ввода)
- **Командные чаты** — общение внутри платформы без перехода в Telegram
- **PWA / мобильная версия** — оптимизация под телефон
- **Реальная оплата Captain PRO** — подключение эквайринга (ЮKassa / Tinkoff Pay)
- **Реферальная механика** — приведи друга, получи продление PRO

**Целевые метрики фазы:**
- 500+ профилей
- 10+ платящих Captain PRO
- 5+ команд через платформу на реальных хакатонах

---

## Фаза 4 — Масштаб
**Июнь 2026+ · ongoing**

**Цель: выйти на B2B и стать инфраструктурой для IT-сообщества**

- **B2B для организаторов** — интеграция SkillHub в процесс регистрации хакатона
- **Верификация достижений** — значки за победы, сертификаты курсов
- **Матчинг по проектам** — не только люди ищут команды, но и проекты ищут людей
- **API для вузов** — студенческие рейтинги внутри факультета
- **Партнёрства** — Яндекс, VK, Тинькофф (спонсоры студенческих хакатонов)
