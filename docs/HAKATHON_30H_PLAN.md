## Краткая сводка проекта

| Параметр | Значение |
|----------|----------|
| **Название** | SkillHub |
| **Слоган** | «Один профиль — всё IT-сообщество» |
| **Проблема** | Хаос в чатах, «кот в мешке», капитаны тратят 4–6 ч на отсев |

| **Монетизация** | Free + PRO Captain 299 ₽/мес + B2B для организаторов |

# Риски и митигация

| Риск | Митигация |
|------|-----------|
| YandexGPT API недоступен или долго отвечает | Fallback: случайный балл 60–90 для демо; кэшировать ответы |
| GitHub OAuth не успеваем | Telegram Login Widget как запасной вариант (проще) |
| Не успеваем деплой | Railway/Render — деплой за 5 минут по GitHub |
| Разные версии Node/npm | `nvm use` или `.nvmrc` с LTS |
| Конфликты в Git | Работать в ветках, мержить часто |
