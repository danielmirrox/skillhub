# Блок 2 — Заметки Дени

## Дата
2026-03-21

## Цель блока
Проверить профильный поток и скоринг так, чтобы это стабильно работало в демо.

## Что сделано
- Подготовлены тестовые профили для разных ролей и уровней.
- Проверен поток: обновление профиля -> скоринг -> история скоринга.
- Проверен импорт GitHub через `githubData` для улучшения подсказок по стеку и проектам.
- Добавлены инструкции по прогону в `scripts/check-score-cases.md`.

## Что проверяли
1. `PUT /api/v1/profile`
2. `POST /api/v1/profile/score`
3. `POST /api/v1/profile/import-github`
4. `GET /api/v1/profile/score/history`

## Ожидаемый результат
- Скоринг возвращает корректные поля (`score`, `grade`, `roleLabel`).
- История обновляется после каждого прогона.
- Импорт GitHub улучшает данные профиля для следующего скоринга.
- Нет неожиданных ошибок `500` на валидных запросах.

## Риски
- Ограничение частоты запросов может вернуть `429` при частых повторах.
- Без внешних AI-интеграций часть score может оставаться демо-логикой.
